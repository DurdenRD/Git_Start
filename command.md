### Быстрые команнды из семинара

# Folders and files

**pwd** - текущая папка
**cd** -  переейти в папку

**touch** - создать файл
**ls** - просмотреть содержание папки
**ls -a** - какие скрытые папки


**git rm** удалить очередность

**wq** - сохранить выйти

# Инструкция vim

**vim** - внести изменени в файл
**shift + i** - редактировать
**esc** - выход
**shift** + : - вести команду
**wq** - сохранить выйти
